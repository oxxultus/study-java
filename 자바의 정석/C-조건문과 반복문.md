# 조건문
## if 문
> 만일 조건식의 참이면 괄호안의 문장들을 수행하라.
```java
if(조건식){
	// 조건식이 참일 때 수행 될 문장들을 적는다.
}
```
> [!CAUTION]
> 자바에서 조건식은 반드시 true 혹은 false이어야 한다는 것을 잊지 말자.
> 만일 블럭 내의 문장이 하나뿐 일 때는 괄호{}를 생략할 수 있다.(가능하면 비선호)
### 자주사용되는 조건식
|조건식|조건식이 참일 조건|
|--|--|
|90 <= x && x <= 100|정수x가 90 이상 100 이하일 때|
|x < 0 \|\| x > 100|정수 x가 0보다 작거나 100보다 클 때|
|x % 3 == 0 && x % 2 != 0|정수 x가 3의 배수이지만, 2의 배수는 아닐 때|
|ch == 'y' && ch =='Y'|문자 ch가 'y'또는 'Y' 일때|
|ch = ' ' \|\| ch = '\t' \|\| ch == '\n'|문자 ch가 공백이거나 탭 또는 개행 문자일 때|
|'A' <= ch && ch <= 'Z'|문자 ch가 대문자일 때|
|'a' <= ch && ch <= 'z'|문자 ch가 소문자일 때|
|'0' <= ch && ch <= '9'|문자 ch가 숫자일 때|
|str.equals("yes")|문자열 str의 내용이 "yes"일 때(대소문자 구분)|
|str.equalsIgnoreCase("yes")|문자열 str의 내용이 "yes"일 때(대소문자 구분안함)|

## if-else 문
> 만일 조건식의 참이면 괄호안의 문장들을 수행하라.
```java
if(조건식){
	// 조건식이 참일 때 수행 될 문장들을 적는다.
} else{
	// 조건식이 거짓일 때 수행될 문장들을 적는다.
}
```
## if-else if문
> 만일 조건식의 참이면 괄호안의 문장들을 수행하라.
```java
if(조건식1){
	// 조건식1이 참일 때 수행 될 문장들을 적는다.
} else if(조건식2){
	// 조건식2이 참일 때 수행 될 문장들을 적는다.
} else if(조건식3){
	// 조건식3이 참일 때 수행 될 문장들을 적는다.
} else{ //생략 가능하지만 보통 작성한다.
	// 모든 조건식이 거짓일 때 수행될 문장들을 적는다.
}
```
## 중첩 if문
> if문의 블럭{} 내에 또다른 if문을 포함시키는 것이 가능한데 이것을 충접 if문 이라고브르며 중첩의 횟수에는 거의 제한이 없다.
```java
if(조건식){
	// 조건식1의 연산결과가 참일 때 수행될 문장들
	if(조건식2){
		// 조건식1과 조건식2가 모두 참일때 수행될 문장들
	} else{
		// 조건식1은 참이고, 조건식2는 거짓일때 수행될 문장들
	}
} else{
	// 조건식1이 거짓일 때 수행되는 문장들
}
```

## switch문
> 먼저 조건식을 계산한다
> 조건식의 결과와 일치하는 case문으로 이동한다.
> 이후의 문장들을 수행한다.
> break문이나 switch문의 끝을 만나면 switch문 전체를 빠져나간다.
```java
switch(조건식){
	case 값1:
		// 조건식의 결과가 값1과 같은경우 수행될 문장들
		break;
	case 값2:
	case 값3:
		// 조건식의 결과가 값2이거나 값3과 같은경우 수행될 문장들
		// 해당방식으로 두개의 값을 하나의 결과로 통일시킬 수 있다.
		break;
	default:
		// 조건식의 결과와 일치하는 case문이 없을때 수행될 문장
}
```
> [!CAUTION]
> switch문의 조건식 결과는 정수 또는 문자열 이어야 한다.
> case문의 값은 정수 상수만 가능하며, 중복되지 않아야 한다.
### switch문의 중첩
> 중첩 switch문의 사용 예제
```java
switch(gender){
	case '1': case '3': // 두개의 경우를 하나의 결과로 만들수 있다.
		switch(gender){
			case '1';
				System.out.printIn("당신은 2000년 이전에 출생한 남자");
				break;
			case '3';
				System.out.printIn("당신은 2000년 이후에 출생한 남자");
				break;
		}
		break; // switch문에서 나오는 break구문을 빼먹지 않도록 주의하자
	case '2': case '4':
		switch(gender){
			case '2';
				System.out.printIn("당신은 2000년 이전에 출생한 여자");
				break;
			case '4';
				System.out.printIn("당신은 2000년 이후에 출생한 남자");
				break;
		}
		break; // switch문에서 나오는 break구문을 빼먹지 않도록 주의하자
	default:
		System.out.printIn("유요하지 않은 값");
}
```
# 반복문
## for문
> for문의 기본형
```java
for(초기화;조건식;증감식){
	// 조건식이 참일 때 수행될 문장
}
코드의 수행방식은 초기화를 수행한 뒤 조건식, 조건식이 참일 때 수행될 문장, 증감식 순서로 진행되며 조건식이 거짓이되면 반복은 종료된다.
```
> for문의 사용예제
```java
// 	1부터 5까지 1식 증가하면서 반복한다.
for(int i=1;i<=5;i++){
	// 조건이 참일 때 수행될 문장
}
```
### 중첩 for문
> if문 안에 또다른 if문을 넣을 수 있는 것처럼, for문 안에도 또 다른 for문을 포함 시키는 것이 가능하다.
> 중첩 for문의 사용 예제
```java
for(int i = 0; i < 5; i++ ){ // 반복문에서 조건식이 참이라면
	for(int j = 0; j<=10; j++){ // 해당 반복문을 실행한다.
		// 조건이 참일때 수행될 문장
	}
}
```
## while문
> for 문에 비해 while문은 구조가 간단하다
```java
while(조건식){
	// 조건식의 연산결과가 참인 동안, 반복됢 문장들을 적는다.
}
```
> while문의 사용 예제
```java
int i = 0; // 초기화
while(i<=10){ // 조건식
	System.out.printIn(i);
	i++; // 증감식
}
```
> [!CAUTION]
> while문의 조건식은 생략이 불가능하다.

## do-while문
> while문과 반대로 블럭을 먼저 수행한 후에 조건식을 평가한다.
```java
do{
	// 일단 한번 수행한다.
	// 조건식의 연산결과가 참인 동안, 반복될 문장들을 적는다.
}while(조건식); // 세미콜론을 잊지 않도록 주의한다.
```
> while문의 사용 예제
```java
int i = 0; // 초기화
while(i<=10){ // 조건식
	System.out.printIn(i);
	i++; // 증감식
}
```
> [!CAUTION]
> while문의 조건식은 생략이 불가능하다.

## break문
> 자신이 포함된 가장 가까운 반복문을 벗어난다. 주로 If문과 같이 사용된다.
```java
while(true){ // 무한 반복문
	if(sum > 100) break; // sum이 100보다 커지면 반복을 종료한다
	++i; // 전위 증가 연산자
	sum += i; // 대입 연산자
}
```
## continue문
> 반복문 내에서만 사용할 수 있으며, 반복이 진행되는 도중에 continue문을 만나면 반복문의 끝으로 이동하여 다음 반복으로 넘어간다.
```java
for(int i = 0;i <= 10;i++){
	if(i%3 == 0) continue; // 3의 배수 출력을 제한, 아래 모든 구문을 건너뛰고 반복을 재시작한다.
	System.out.printIn(i);
}
```

## 이름 붙은 반복문 (**중요**)
> break문은 근접한 단 하나의 반복문만 벗어날 수 있기 때문에, 여러 개의 반복문이 중첩된 경우에는 break문으로 중첩 반복문을 완전히 벗어날 수 없다.
```java
반복문의이름 : for(;;) 형식이다.

Loop : for(int i = 2;i <= 9; i++){
			for(int j = 1;j <=9; i++){
				if(j==5){ 
					break; // 일 경우에는 가장가까운 반복분을 벗어나지만
					break Loop; //j가 5일경우 Loop 반복을 종료한다.
				}
			}
		}
```
